# Redes de Computadores 🌐

Notas de aula, resumos e simulações sobre a infraestrutura de redes e comunicação de dados.

## 📖 Tópicos Abordados:
- Modelos OSI e TCP/IP.
- Protocolos de comunicação (HTTP, FTP, DNS, etc).
- Endereçamento IP (IPv4 e IPv6).
- Topologias de rede e equipamentos (Hubs, Switches, Roteadores).
